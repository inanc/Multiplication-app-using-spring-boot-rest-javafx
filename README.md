#Multiplication spring boot rest javafx client

####Install
$ git clone https://github.com/inanc/Multiplication-spring-boot-rest-javafx-client.git


#### How to build
##### Mac and Linux
```bash
./gradlew clean build
```
##### Windows
```bash
> gradlew clean build
```
